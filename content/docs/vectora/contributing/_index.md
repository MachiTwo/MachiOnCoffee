Vectora é um projeto open source (MIT) e recebe contribuições de toda a comunidade. Este guia cobre como reportar bugs, sugerir features, contribuir código e melhorar documentação.

## Tipos de Contribuição

| Tipo              | Esforço | Impacto | Como                                                           |
| ----------------- | ------- | ------- | -------------------------------------------------------------- |
| **Bug Reports**   |         |         | [→ Issues](https://github.com/Kaffyn/Vectora/issues)           |
| **Docs**          |         |         | [→ Guidelines](./guidelines.md)                                |
| **Code Features** |         |         | [→ Roadmap](./roadmap.md)                                      |
| **Bug Fixes**     |         |         | Fork + Pull Request                                            |
| **Tests**         |         |         | src/**tests**/                                                 |
| **Feedback**      |         |         | [→ Discussions](https://github.com/Kaffyn/Vectora/discussions) |

## Quick Start

1. **Fork** o repositório
2. **Clone** sua cópia: `git clone https://github.com/seu-usuario/vectora.git`
3. **Crie uma branch**: `git checkout -b feat/sua-feature`
4. **Commit**: `git commit -am "Add feature"`
5. **Push**: `git push origin feat/sua-feature`
6. **Pull Request** no repositório original
